# 6006CEM Machine Learning Coursework
This repository contains the source code for my 6006CEM Machine Learning and Related Applications coursework.

## Contents
- `classification.ipynb` — supervised learning (logistic regression & MLP) for diabetes prediction.
- `regression.ipynb` — linear regression & random forest for house price prediction.

- ## Datasets
- Diabetes Prediction Dataset (Mustafa Tz, 2023) — Kaggle URL:  
  https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset

- House Prices: Advanced Regression Techniques — Kaggle URL:  
  https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data
